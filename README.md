1. **Clone the Repository:**
   ```bash
   git clone https://github.com/karanDevX11/Django-Projects.git
   ```

   ```bash
   cd django-projects
   ```
3. **Create a Virtual Environment:**
   ```bash

   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

